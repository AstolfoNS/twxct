# jxufe-twxct

Top-level workspace for the TWXCT project.

## Structure

```text
jxufe-twxct/
├── backend/    # Java / Spring Boot services
├── frontend/   # Vue 3 / Vite applications
└── README.md
```

## Quick Start

### Backend

```powershell
cd backend
```

### Frontend

```powershell
cd frontend
```

## Notes

- This repository is a workspace that aggregates backend and frontend code.
- Backend and frontend can be built and deployed independently.
